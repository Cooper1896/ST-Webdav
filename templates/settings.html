WebDAV File Manager
Server plugin is required for WebDAV operations.

To complete the setup, run the following command in your SillyTavern directory:

Windows (PowerShell): 
cd <SillyTavern目录>
.\data\<user>\extensions\third-party\sillytavern-webdav\install.ps1
macOS / Linux: 
cd <SillyTavern目录>
bash data/<user>/extensions/third-party/sillytavern-webdav/install.sh

Then restart SillyTavern.

Disconnected
Password is never stored in browser. It is encrypted on the server only.
